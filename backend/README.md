# AI Coding Interview Simulator

An AI-powered full stack web application that helps developers practice coding interviews. Built with Python Flask, React.js, MySQL, and integrated with AI to generate questions and evaluate code.

## Features
- User registration and login with JWT authentication
- AI generates unique coding questions based on topic and difficulty
- In-browser code editor to write your answer
- AI evaluates your code and returns score out of 100, feedback, and ideal solution
- Dashboard tracking total attempts, average score, and pass rate
- Full attempt history saved per user

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React.js |
| Backend | Python Flask |
| Database | MySQL |
| AI | OpenRouter API |
